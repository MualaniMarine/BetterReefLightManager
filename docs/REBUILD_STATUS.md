## 3. 当前可用流程

### AP 直连控制

适用于设备已开热点或需要热点直连控制的情况：

1. 进入网络配置
2. 开启 `AP 模式`
3. 扫描 `K7_Pro` 热点
4. 连接热点
5. 自动带入网关 IP
6. 进入控制调试直接连接设备

### 局域网控制

适用于设备已经进路由器的情况：

1. 进入网络配置
2. 切到 `局域网模式`
3. 扫描同网段设备
4. 带入设备 IP
5. 进入控制调试连接设备

## 5. 当前确认的重要业务结论

- `SPS / LPS / SL` 是三组预设时间-亮度数据，不是运行模式
- `自动 / 手动` 才是运行模式
- `演示模式` 是单独的特殊状态
- `AP 模式` 下控制页自动带入的是路由器 / 网关 IP
- `局域网模式` 下带入的应是扫描到的设备 IP，控制页不应覆盖成路由器 IP
